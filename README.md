## Kelly's Hello World App

### App Description
This app allows you to change the background color of the app with the "Change View Color" button and change the color of the text with the "Change Text Color" button. In addition a user can type in text with the "Change Text String" button and change the default "Hello from Kelly" view on the screen.If there is no text in the text field when the "Change Text String" button is pressed is "Goodbye ðŸ‘‹" is automatically added. Lastly if you click on the background you are able to reset the view to the default setting.
